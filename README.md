## Getting started

#### 1. Clone the repo

```sh
$ git clone https://github.com/deepeshsharmaofficial/react-portfolio
$ cd react-portfolio/
```
#### 2. Install dependencies

```sh
$ npm i
```

#### 3. Update data

- Update the [`index.js`]() in `constants` folder

#### 4. Run the project

```sh
$ npm run dev
```
